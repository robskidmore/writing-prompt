A simple writing prompt plugin for Wordpress.

== Description ==

Get the daily writing prompt from [The Write Practice](http://thewritepractice.com) right in your Wordpress admin area. Is there a better way to facilitate consistent and creative writing? I submit that there is not.

== Installation ==

1. Upload `writing-prompts` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open up a new post
4. Write like crazy
